Add a fixed price product

In the admin menu click on Add content and you will be able to add a new Product display. A Product display is a fixed price product.

On the following page you will be able to add the Title, Description and Price and one or more imags for the new item. Then click the Add product button.

Select the relevant Product cvategeory. This value determines where the new item will appear in the main menu.

Note: Only items that have a Product categeory value of Additional Products will atrract the flat rate delivery fee.
